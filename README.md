# Text to 3D Web Application

A browser-based project that converts text prompts into 3D visualizations using Three.js.

## Features
- Text-based UI
- 3D model rendering (GLB / glTF)
- Orbit controls for interaction
- Clean web interface

## Tech Stack
- HTML, CSS, JavaScript
- Three.js
- GLTFLoader

## How to Run
1. Open `index.html` in a browser
2. Enter a text prompt
3. View the generated 3D model
